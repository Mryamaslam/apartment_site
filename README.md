# City Scape Faletti’s Service Apartment

This repository contains the source code for the **City Scape Faletti’s Service Apartment** website. It is a modern, responsive website designed to showcase luxury service apartments, amenities, and floor plans.

## Project Overview

The website serves as a digital brochure for potential investors and residents, providing detailed information about the project's offerings, location, and leadership.

### Key Features

*   **Responsive Design:** Built with Bootstrap 5 to ensure a seamless experience across all devices (mobile, tablet, desktop).
*   **Dynamic Hero Section:** Features a custom split-screen video layout on the homepage with looping playlists.
*   **Apartment Showcase:** Detailed pages for various apartment categories (Studio, 1 Bed, 2 Bed, etc.) with image galleries.
*   **Gallery:** A comprehensive photo gallery of the project's interior and exterior.
*   **Contact & Location:** Easy-to-access contact information and location details.
*   **Leadership Team:** Profiles of the project's key leadership figures.

## Technologies Used

*   **HTML5**
*   **CSS3** (Custom styling + Bootstrap 5)
*   **JavaScript** (ES6+)
*   **Bootstrap 5** (Framework)
*   **Font Awesome** (Icons)

## Project Structure

*   `index.html`: Main landing page.
*   `apartments.html`: Listing of available apartment types.
*   `amenities.html`: Details about project amenities.
*   `gallery.html`: Image gallery.
*   `location.html`: Map and location information.
*   `contact.html`: Contact form and details.
*   `builders.html`: Information about the developers and leadership team.
*   `assets/`: Contains images and video resources.
*   `css/`: Stylesheets (`style.css`).
*   `js/`: JavaScript files (`script.js`, `data.js`).

## Getting Started

To view the website locally:

1.  Clone this repository.
2.  Open the `index.html` file in your preferred web browser.
